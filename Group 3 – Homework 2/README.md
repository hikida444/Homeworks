# Homework 3 — enum/struct + задачи со строками

Один консольный проект, в котором последовательно выполнены упражнения по перечислениям (`enum`), структурам (`struct`), типам данных и базовым задачам со строками/математикой.

> [!IMPORTANT]
> Файл содержит несколько независимых частей, которые выполняются подряд в `Main()`.

## Что реализовано

### Упражнение 3.1 — AccountType
- Перечисление `AccountType` (Current/Savings).
- Создание переменной этого типа и вывод значения.

### Упражнение 3.2 — BankAccount
- Структура `BankAccount` с полями:
  - `Number` — номер счёта
  - `Type` — тип счёта (`AccountType`)
  - `Balance` — баланс (`decimal`)
- Создание структуры, заполнение полей и вывод данных.

### Домашнее задание 1 — Employee/University
- Перечисление `University` (KGU/KAI/KHTI).
- Структура `Employee` с полями `Name` и `Univ`.
- Создание работника и вывод информации.

### Задача 1 — Min/Max типов
- Вывод минимальных и максимальных значений для `int`, `double`, `decimal`.

### Задача 2 — Переключение регистра
- Читает строку и меняет регистр букв:
  - строчные → заглавные
  - заглавные → строчные
- Остальные символы не меняются.

### Задача 3 — Поиск подстроки
- Читает строку и подстроку.
- Считает количество вхождений подстроки с помощью `IndexOf` в цикле.

### Задача 4 — Duty Free (расчёт бутылок)
- Считает экономию на одной бутылке по скидке.
- Вычисляет, сколько бутылок нужно купить, чтобы “отбить” стоимость отпуска.

### Задача 5 — Студенты и напитки
- Структуры:
  - `Drink` (название, процент алкоголя)
  - `Student` (ФИО, id, дата рождения, категория, объём, тип напитка)
- Создаётся 5 студентов с разными напитками.
- Считается общий объём выпитого и общий объём чистого алкоголя.
- Для каждого студента рассчитываются индивидуальные показатели и доля от общих значений.

> [!NOTE]
> В некоторых задачах используется ввод с клавиатуры (строки/числа), поэтому при запуске программа будет ждать ввод в консоли.

## Как запустить
1. Создай консольный проект C#.
2. Замени содержимое `Program.cs` на код из файла.
3. Запусти из IDE или через `dotnet run`.
