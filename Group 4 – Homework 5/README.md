# Задачи 6.x — строки, массивы, коллекции

В этой папке собраны шесть задач: подсчёт гласных/согласных, умножение матриц (массивы и LinkedList), средние температуры (2D‑массив и Dictionary).

## Содержимое

- 6.1 — Подсчёт гласных и согласных в текстовом файле (char[])
- 6.2 — Умножение двух матриц (двумерные массивы)
- 6.3 — Средние температуры по месяцам (двумерный массив)
- 6.1 HW — Подсчёт гласных и согласных (List<char>)
- 6.2 HW — Умножение матриц (LinkedList<LinkedList<int>>)
- 6.3 HW — Средние температуры по месяцам (Dictionary<string, double[]>)

## Как запускать

- Каждое решение — независимый консольный файл.  
- Откройте проект, временно замените содержимое `Program.cs` кодом нужного файла (или подключите файл как точку входа), затем запустите проект.
