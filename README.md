# C# Homeworks 

Репозиторий содержит домашние задания по **программированию на C#** от студентов группы — собраны в одном месте для удобного просмотра и сравнения решений.

> [!IMPORTANT]
> Каждая папка студента — независимый набор решений. Код можно запускать отдельно через Visual Studio, Rider или `dotnet run`.
