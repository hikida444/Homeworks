# Интерфейсы и полиморфизм (ICipher, IFigure)

Папка содержит две задачи:

1. **ICipher** — интерфейс для шифрования/дешифрования строк и две реализации:
   - `A` — шифр со сдвигом буквы на +1 в русском алфавите.
   - `B` — «зеркальный» шифр: каждая буква заменяется на симметричную относительно конца алфавита.

2. **IFigure / Figure** — работа с геометрическими фигурами:
   - Общий интерфейс `IFigure` и абстрактный базовый класс `Figure`.
   - Конкретные фигуры: `Point`, `Circle`, `Rectangle`.
   - Демонстрация полиморфизма через массив `IFigure[]` и общие операции: движение, смена цвета, вывод информации, вычисление площади.

## Файлы

- `CipherInterfaces.cs` — интерфейс `ICipher` и классы `A`, `B` (реализации шифров).
- `FiguresPolymorphism.cs` — интерфейс `IFigure`, абстрактный класс `Figure`, классы `Point`, `Circle`, `Rectangle` и `Program.Main`.

## Принцип

> Один интерфейс — много реализаций:  
> - Для строк — `ICipher` + разные шифраторы.  
> - Для фигур — `IFigure` + разные геометрические фигуры с общим набором операций.

## Как запускать

- Для проверки шифров (`CipherInterfaces.cs`) можно написать отдельный `Main` или использовать существующий проект и вызвать `ICipher`‑реализации вручную.
- Для фигур достаточно сделать `FiguresPolymorphism.cs` основным `Program.cs` и запустить консольное приложение.
