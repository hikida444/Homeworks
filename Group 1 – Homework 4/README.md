# Домашние задания по C#

> [!IMPORTANT]
> Каждая программа — самостоятельное консольное приложение. Код можно скопировать в `Program.cs` и запустить из Visual Studio, Rider или через `dotnet run`.

## 1. MaxNumber.cs — Максимум из двух чисел

Программа запрашивает два целых числа и выводит наибольшее из них через метод `Maximum(x, y)`, который использует `Math.Max`.

---

## 2. SwapNumbers.cs — Обмен двух чисел

Считывает два числа, передаёт их в метод `Swap(ref a, ref b)` для обмена местами через временную переменную, и выводит результат до и после.

---

## 3. FactorialWithCheck.cs — Факториал с проверкой переполнения

Вычисляет факториал целого числа и отслеживает переполнение типа `long` с помощью блока `checked` и перехвата `OverflowException`. Метод `Factorial(int n, out long result)` возвращает `true` при успехе и `false` при переполнении.

---

## 4. RecursiveFactorial.cs — Рекурсивный факториал

Вычисляет факториал рекурсивно: `Factorial1(x) = x * Factorial1(x - 1)`, базовый случай — `x == 1`. Работает корректно до 20 включительно.

> [!NOTE]
> Выше 20 происходит переполнение типа `long`.

---

## 5. GCDCalculation.cs — НОД двух и трёх чисел

Находит наибольший общий делитель двух или трёх чисел через алгоритм Евклида. Использует перегрузку методов: `nod(a, b)` и `nod(a, b, c)`.

---

## 6. FibonacciRecursive.cs — Рекурсивная функция Фибоначчи

Вычисляет n-е число Фибоначчи рекурсивно: `F(n) = F(n-1) + F(n-2)`, базовые случаи `F(1) = F(2) = 1`. Рекурсия без оптимизации (экспоненциальная сложность).

---

## 7. ArrayElementSwap.cs — Обмен элементов в массиве

Создаёт массив из 20 случайных чисел от 1 до 98, выводит его, запрашивает два числа и меняет их местами. Использует `Array.IndexOf` для поиска индексов и проверяет наличие элементов в массиве.

---

## 8. ParamsExample.cs — Работа с параметрами

Демонстрирует использование `ref`, `out` и `params`: метод `MassiveWork` вычисляет сумму, произведение и среднее арифметическое переменного количества аргументов.

**Реализация:**
- `ref long proiz` — передаёт произведение по ссылке
- `out double average` — возвращает среднее через выходной параметр
- `params int[] numbers` — принимает переменное количество аргументов

---

## 9. DigitalArt — Цифры в виде ASCII-рисунков

Программа считывает строку, проверяет её на команды выхода (`exit`, `закрыть`) или преобразует в число от 0 до 9 и выводит соответствующую цифру псевдографикой через `switch`. Обрабатывает исключения `ArgumentOutOfRangeException` и `FormatException`, при ошибке меняет цвет текста на красный.

---

## 10. GrandpaSwearCheck.cs — Проверка матерных слов

Программа создаёт массив структур `Grandpa` с уровнями ворчливости (`enum GrumpinessLevel`) и фразами. Метод `CheckSwearing` проверяет фразы на наличие матерных слов и начисляет «фингалы» — увеличивает счётчик `Bruises`.

**Реализация:**
- `struct Grandpa` с полями `Name`, `Grumpiness`, `Phrases[]`, `Bruises`
- `enum GrumpinessLevel { Low, Medium, High }`
- Статический метод `CheckSwearing(ref Grandpa, params string[])` для проверки фраз (регистронезависимо через `.ToLower()`)

## Как запустить решения

> [!NOTE]
> Все файлы — обычные консольные программы C#. Каждую можно запускать отдельно.

### Вариант 1: через Visual Studio / Rider / VS Code

1. Создайте новый проект «Консольное приложение» на C#.
2. Откройте файл `Program.cs` и **замените** его содержимое кодом из нужного файла (например, `MaxNumber.cs`).
3. Запустите проект (кнопка ▶ или клавиша F5).

### Вариант 2: через .NET CLI

1. Установите .NET SDK.
2. В папке, где лежит `.csproj` проекта, выполните в терминале команду:

dotnet run
