# Homework 4 — задачи с датами, switch и массивами

Один консольный файл с набором заданий по C#: работа с массивами, `switch`, строками и датами. Задания оформлены как локальные функции внутри `Main()`.

> [!IMPORTANT]
> Для запуска конкретного задания нужно раскомментировать вызов нужного метода в `Main()` (например `Task41();`, `Task2();` и т.д.). Локальные функции доступны только внутри `Main()`. [web:154]

---

## Список заданий

### Task4.1 — День года → дата
Конвертирует порядковый номер дня в году (1–365) в «день + месяц» с помощью массива дней в месяцах и последовательного вычитания.

### Task4.2 — День года → дата (с проверками)
То же, что Task4.1, но с дополнительной проверкой корректности вводимого значения.

### TaskDz4.1 — Учет високосного года
Расширение 4.1/4.2: дополнительно запрашивает год и учитывает високосность (использует разные массивы для февраля: 28 или 29 дней).

---

### Task1 — Проверка последовательности на возрастание
Генерирует массив из 10 случайных целых чисел и проверяет, является ли последовательность строго возрастающей; если нет — выводит номер первого элемента, который нарушает порядок.

### Task2 — Достоинство карты по номеру
По номеру карты (6–14) определяет достоинство (шестерка…туз) через `switch`-выражение и обрабатывает ошибки ввода.

### Task3 — Напиток по профессии
Принимает строку (профессию) и возвращает “рекомендуемый напиток” через `switch` по строке (в нижнем регистре).

### Task4 — День недели по номеру
По числу 1–7 выводит день недели на русском, используя перечисление `DayOfWeek` и вспомогательный метод `GetDayName()`.

### Task5 — Подсчет кукол в массиве
Есть массив строк с названиями игрушек; считает, сколько раз встречаются `"Hello Kitty"` или `"Barbie doll"`.

---

## Как запустить

1. Создай консольный проект C#.
2. Замени содержимое `Program.cs` на код из этого файла.
3. В `Main()` раскомментируй вызов нужной задачи (например `Task41();`).
4. Запусти из IDE или через `dotnet run`.

> [!NOTE]
> Некоторые задания ожидают ввод с клавиатуры (строки/числа), поэтому при запуске программа будет ждать ввод в консоли.
